Funcionalidades

Características del proyecto

🔒

HTTPS con certificado x.509

Certificado autofirmado generado automáticamente con OpenSSL mediante Ansible.

🔑

Autenticación básica

Acceso protegido mediante htpasswd gestionado por Ansible sin intervención manual.

💾

Almacenamiento persistente

PersistentVolumeClaim en AKS con storage class managed-csi para Redis.

🌐

Acceso público

Service de tipo LoadBalancer que expone la aplicación con IP pública en Azure.

📦

Registry privado

Imágenes almacenadas en ACR con integración nativa AKS mediante rol AcrPull.

🤖

100% automatizado

Despliegue completo desde cero con un único comando sin intervención manual.
